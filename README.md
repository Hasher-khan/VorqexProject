# Vorqex Project - Portfolio Option Page

A modern, glassmorphic portfolio showcase page built for **Vorqex Project**—a creative digital agency. This page showcases various projects including Web Apps, Branding, E-Commerce, Web Design, and AI Automations.

## Tech Stack
- **HTML5** & **Tailwind CSS** (via CDN with custom configuration)
- **Alpine.js** (for responsive tabs, interactive modals, and state management)
- **Google Fonts (Outfit)**

## Features
- **Responsive Navigation**: Sticky header with a glassmorphism effect (`glass-nav`) and mobile dropdown menu.
- **Dynamic Project Filtering**: Filter projects seamlessly by categories (All, Web App, Branding, E-Commerce, Web Design, AI Automation, AI Agent) using Alpine.js.
- **Glassmorphic Cards**: Premium card design with glow hover animations (`glow-card`) and hover scales.
- **Interactive Case Study Modal**: Multi-column responsive layout displaying project details, tech stack tags, and external site links with smooth transition effects.
- **Polished Footer**: Organized sub-navigation links, brand pitch, and legal footer.

## Live Preview
To preview the page, simply open `projects.html` in any web browser.
